# Week 4 Python Practice

Welcome to the Sejong Advanced Python Week 4 practice session. This repository contains the practice file for this week.

## Topic

Control statements: `for`/`while` loops, `break`, `continue`, nested loops.

## How to Get Started

### 1. Clone the Repository
Clone this repository to your local machine using the following command:
```bash
git clone https://github.com/sejong-advanced-python/ta-practice-students-week4.git
```
*(Note: Replace the URL with your actual personal repository link if provided by the TA.)*

### 2. Complete the Exercises
Open `week04_practice.py` in your preferred editor (VS Code, PyCharm, etc.) and complete the tasks as instructed during the session.

### 3. Push Your Changes
Once you have finished the exercises, commit and push your changes:
```bash
git add week04_practice.py
git commit -m "Complete week 4 practice"
git push origin main
```

## How to See Results
- Run the script locally to check for any errors:
  ```bash
  python week04_practice.py
  ```
- If there are automated tests provided, follow the instructions given by the TA to run them.

## Common Problems & Solutions

- **Problem: Permission Denied during clone/push**
  - *Solution:* Ensure you have set up your SSH keys or personal access token (PAT) for GitHub correctly.

- **Problem: File not found when running the script**
  - *Solution:* Make sure you are in the correct directory (`ta-practice-students-week4`) before running the command.

- **Problem: Merge conflicts**
  - *Solution:* If you are working on a shared repository, always `git pull` before starting your work and before pushing.

- **Problem: Python version mismatch**
  - *Solution:* Ensure you are using Python 3.8 or higher. Check your version with `python --version`.
